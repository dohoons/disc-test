# DISC 성격 유형 검사

자신의 성격 유형을 이해하고 팀원들과의 시너지를 분석하는 DISC 성격 검사 웹 애플리케이션입니다.

## 주요 기능

- **DISC 성격 검사**: 24개 질문을 통해 D(지배), I(영향), S(안정), C(신중) 4가지 성격 유형 분석
- **시각적 결과**: 레이더 차트와 행동 지표로 직관적인 결과 제공
- **팀 시너지 분석**: 팀원과의 성격 조합 분석 및 협업 팁 제공
- **결과 공유**: URL을 통해 검사 결과 간편 공유

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

### 빌드

```bash
npm run build
```

## 기술 스택

- React 18 + Vite + TypeScript
- Tailwind CSS
- Chart.js
- React Router
